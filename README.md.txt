# Sentiment Analysis Project

## Aim
Create a sentiment analysis tool that classifies text as Positive, Negative, or Neutral.

## Description
This project uses Python and TextBlob to analyze the sentiment of user-provided text. It includes a simple Tkinter GUI for input and displaying results.

## Technologies Used
- Python
- TextBlob
- Tkinter

## How to Run
1. Install required libraries: `pip install -r requirements.txt`
2. Run the app: `python app.py`
